## HOW TO USE THIS SCRIPT

### Step 1 — Choose your tool
- **HeyGen** (heygen.com) — Best avatar realism, supports custom avatars
- **Synthesia** (synthesia.io) — Best for corporate/professional look
- **InVideo AI** (invideo.ai) — Easiest to add stock footage B-roll

### Step 2 — Set up your project
1. Create a new video project
2. Select a professional avatar (business casual or formal)
3. Set background: dark blue gradient or abstract tech/data theme
4. Upload AuraQubes logo for intro/outro

### Step 3 — Paste the script
- Copy the avatar dialogue lines only (the `> "..."` text)
- Remove the `[GRAPHIC]` and `[ON-SCREEN]` stage directions from the spoken text
- Add lower-third text and graphics via the tool's text/graphics editor

### Step 4 — Add music
- Use a subtle, professional background track (ambient/corporate)
- Keep volume at 15–20% so the voice stays dominant

### Step 5 — Export
- Export at 1080p (16:9 aspect ratio)
- Save as MP4 for website embed
- Place the file in: `assets/auraqubes-company.mp4` to auto-populate the existing video player on the home page
